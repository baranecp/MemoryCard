# MemoryCard

![Preview](preview.png)

A simple memory card matching game built with React + Vite.

Overview
--------
MemoryCard is a lightweight browser game where players flip cards to find matching pairs. It's implemented with React and bundled with Vite.


Getting started
---------------
1. Clone the repo
   ```
   git clone https://github.com/baranecp/MemoryCard.git
   cd MemoryCard
   ```

2. Install dependencies
   ```
   npm install
   ```

3. Run the development server
   ```
   npm run dev
   ```
